# 다른 컴퓨터 시작 가이드 (복붙용)

## 1) 처음 1회 설정
### 1-1. Git 설치 확인
```bash
git --version
```

### 1-2. Flutter 설치 확인
```bash
flutter --version
```

## 2) 프로젝트 받기
원하는 폴더에서:
```bash
git clone https://github.com/HugoSeong0721/test-mvp.git
cd test-mvp
```

## 3) 실행
```bash
flutter pub get
flutter run -d chrome
```

## 4) 작업 후 저장(업로드)
```bash
git add .
git commit -m "작업 내용"
git push
```

## 5) 다른 PC에서 한 작업 가져오기
이미 `test-mvp` 폴더가 있는 컴퓨터에서:
```bash
cd test-mvp
git pull
```

## 6) 자주 쓰는 최소 명령어
```bash
git status
git pull
git add .
git commit -m "메시지"
git push
```

## 7) 충돌/에러가 나면
아래 2개 출력 결과를 복사해서 저한테 보내주세요:
```bash
git status
git branch -vv
```
